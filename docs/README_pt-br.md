# knight_game

Jogo simples de coleta de moedas feito com SDL3.

## Dependências
* SDL3
* SDL3_image
* SDL3_ttf
* SDL3_mixer

## Instalação

Clone este repositório:

```
https://github.com/5tack-s/knight_game/
```

Depois de instalar todas as dependências, execute o seguinte em knight_game/ para compilar:
```
cmake -S . -B build/
cmake --build build
```
