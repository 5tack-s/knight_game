# knight_game

Simple coin collecting game made using SDL3.

## Dependencies
* SDL3
* SDL3_image
* SDL3_ttf
* SDL3_mixer

## Installing

Clone this repository:

```
https://github.com/5tack-s/knight_game/
```

After installing all the dependencies, run the following in knight_game/ to compile:
```
cmake -S . -B build/
cmake --build build
```
